# WeatherBot
Simple telegram weather bot.    
Shows the weather for selected by you region for the current day,
the next day, next three days and for the current week.

[Check out my bot](https://web.telegram.org/#/im?p=@grishasht_bot)

### How to run
- clone this project
- should have Telegram API installed
- create your own bot using [BotFather](https://web.telegram.org/#/im?p=@BotFather)
- create file with yours bot authorization token only and set it in ```commandFilter.py```
- set yours bot commands in ```commandFilter.py```
- run ```python main.py``` in terminal
- open Telegram and use your bot
